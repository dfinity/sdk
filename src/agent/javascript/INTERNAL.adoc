== Development

=== How to build and use locally

In this repo:

[source,bash]
cd src/agent/javascript  # You are here
npm install
npm run bundle
JS_AGENT_PATH=$(pwd)
cd MY_PROJECT_PATH  ## <<<<< Put in the project you want to test.
npm install ${JS_AGENT_PATH}  ## This will likely create a symlink.
