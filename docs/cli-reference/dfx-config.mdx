# dfx config

<MarkdownChipRow labels={["Reference"]} />

The `dfx config` command modifies DFX's configuration. It can also be modified manually (see `dfx info config-json-path`).

## Basic usage

```bash
dfx config <field> [value]
```

When a value is not supplied, dfx will print the current effective value of that field.

## Config fields

| Command                    | Description                                                                                       |
|----------------------------|---------------------------------------------------------------------------------------------------|
| `telemetry [on|off|local]` | Sets whether telemetry is enabled. `local` means dfx will collect the logs but not transmit them. |
